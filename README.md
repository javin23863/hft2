# hft2

Isolated high-frequency trading workspace for research and execution prototyping.

## Scope

This repository is intentionally independent from `quant-x`. It starts as a minimal
greenfield scaffold for strategy research, data adapters, and simulation tooling.

## Quick Start

1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Add your runtime settings from `.env.example` into a local `.env` (never commit secrets).

## Layout

- `src/hft2/` - package source
- `tests/` - tests
- `docs/` - project docs
